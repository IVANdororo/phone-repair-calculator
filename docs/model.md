# Математическая модель

## Калькулятор стоимости ремонта телефона

## Переменные

| Переменная | Описание | Значения |
|------------|----------|----------|
| L | Стоимость работы | зависит от модели и типа ремонта |
| P | Стоимость запчастей | зависит от модели и типа ремонта |
| D | Стоимость диагностики | 500 руб, или 0 для материнской платы |
| S | Стоимость услуг | 0, 300, 500 или 800 руб |
| U | Коэффициент срочности | 1.0, 1.3 или 1.5 |
| C | Скидка | 10% или 15% |
| Итого | Конечная сумма | рубли |

| Промокод | Скидка | Применяется к |

| REMONT26 | 15% | Только стоимость работы |

## Основная формула
Промежуточная сумма = L + P + D + S
Сумма со срочностью = Промежуточная сумма × U
Итоговая сумма = Сумма со срочностью - Скидка
