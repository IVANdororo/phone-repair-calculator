# README.md
Программа предназначена для автоматического расчёта стоимости ремонта мобильных телефонов в сервисном центре. Позволяет определить итоговую цену с учётом:
- модели устройств
- типа неисправности
- стоимсоть запчастей
- срочность выполнения
- дополнительных услуг
- скидок по промокодам

Примечание: можно использовать лишь один промокод за раз

Функциональны требования:
1. Расчет стоимости работ - по таблице в зависимости от модели и типа ремонта
2. Расет стоимости запчастей - вводится пользователем
3. Расче диагносики - если не включена в ремонт, +500 руб.
4. Наценка за срочность  - умножение промежуточной суммы на коэффициент
5. Расчет дополнительных услуг - суммирование выбранных
6. Расчет скидки - по промокоду
7. Валидация - проверка неотрицательности цен, корректность ввода

Пример рассчета
Входные данные:
Модель: iPhone
Ремонт: Замена экрана(работа 2500 руб.)
Запчасти: 5000 руб.
Диагностика: 500 руб.
Срочность: обычный (x1.0)
Услуги: нет
Промокод: Service10

Расчёт:
1. 2500 руб.
2. 5000 руб.
3. 500 руб.
4. Промежуточная сумма: 8000 руб.
5. x1.0
6. 10% скидка = -800 руб.
7. Итого: 7200 руб.
